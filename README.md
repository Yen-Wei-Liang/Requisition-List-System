# 系統晶片之嵌入式即時作業系統材料領用系統

## 一、系統介紹
此系統是一個用於課程材料領用的管理系統，可紀錄每位學員的領用狀況，並加入防呆功能，若未勾選完所有器材領用狀況，系統則無法提供儲存功能。此系統可協助課程主辦單位快速且方便地管理課程材料的發放與歸還。

## 二、使用技術
* Python程式語言
* Tkinter函式庫
